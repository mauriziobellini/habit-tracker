# Habit Ring

An iOS habit tracker app that helps users build daily habits with a beautiful, smooth UI.

## About

The best way to improve yourself is to create small habits. This app allows you to track tasks you want to complete each day in order to form a habit. Built with AI-assisted development to demonstrate rapid app creation.

## Goals

- Beautiful, catching, and smooth iOS UI
- Easy habit creation and daily tracking
- Streak-based motivation system
